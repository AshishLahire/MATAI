# Algebra: Quadratic Equations
- Standard Form: ax² + bx + c = 0
- Roots: x = [-b ± sqrt(b² - 4ac)] / 2a
- Nature of Roots: D = b² - 4ac. D > 0 (Distinct), D = 0 (Real & Equal), D < 0 (Imaginary).
- Vieta's formulas: Sum = -b/a, Product = c/a.

# Algebra: Binomial Theorem
- (a+b)ⁿ = Σ (nCr) aⁿ⁻ʳ bʳ
- General term: Tᵣ₊₁ = nCr aⁿ⁻ʳ bʳ
- Middle term depends on whether n is even or odd.

# Algebra: Permutations & Combinations
- nPr = n! / (n-r)!
- nCr = n! / [r!(n-r)!]
- Circular permutation: (n-1)!

# Calculus: Limits
- lim (x->0) [sin x / x] = 1
- lim (x->0) [tan x / x] = 1
- lim (x->0) [(1+x)^(1/x)] = e

# Calculus: Differentiation Rules
- Power Rule: d/dx(xⁿ) = nxⁿ⁻¹
- Product Rule: (uv)' = u'v + uv'
- Quotient Rule: (u/v)' = (u'v - uv') / v²
- Chain Rule: d/dx[f(g(x))] = f'(g(x))g'(x)

# Calculus: Optimization
- First Derivative Test: Set f'(x) = 0 to find critical points.
- Second Derivative Test: f''(x) > 0 (Local Min), f''(x) < 0 (Local Max).

# Probability: Bayes Theorem
- P(A|B) = [P(B|A)P(A)] / P(B)
- P(B) = Σ P(B|Aᵢ)P(Aᵢ) (Total Probability Theorem)

# Probability: Distributions
- Binomial: P(X=k) = nCk pᵏ (1-p)ⁿ⁻ᵏ
- Poisson: P(X=k) = (λᵏ e⁻λ) / k!

# Probability: Expectation
- E[X] = Σ xᵢ P(xᵢ)
- Var(X) = E[X²] - (E[X])²

# Linear Algebra: Determinants
- |A| = ad - bc (for 2x2)
- Cramer's Rule for solving linear equations.
- Properties: |AB| = |A||B|.

# Linear Algebra: Matrices
- (AB)ᵀ = BᵀAᵀ
- A⁻¹ = (1/|A|) adj(A)
- Eigenvalues: det(A - λI) = 0

# JEE Template: Complex Numbers
- z = a + ib = r(cos θ + i sin θ)
- De Moivre's Theorem: (cos θ + i sin θ)ⁿ = cos nθ + i sin nθ

# JEE Template: Coordinate Geometry
- Line: Ax + By + C = 0
- Distance from point (x₁, y₁) to line: |Ax₁ + By₁ + C| / sqrt(A² + B²)
- Circle: (x-h)² + (y-k)² = r²

# JEE Template: Sequences
- Arithmetic Progression: aₙ = a + (n-1)d, Sₙ = (n/2)(2a + (n-1)d)
- Geometric Progression: aₙ = arⁿ⁻¹, Sₙ = a(rⁿ-1)/(r-1)

# JEE Template: Mistake Patterns
- Improper handling of domain constraints (e.g., denominators not equal to zero).
- Forgetting the constant of integration in indefinite integrals.
- Sign errors during matrix multiplication.
- Incorrect use of inclusive/exclusive boundaries in probability sets.
